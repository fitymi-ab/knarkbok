# knarkbok

Knarkbok / Drug journal web site

Edit the static site as html in `src/main/resources/site`

The index html will be served from the root.

## Build

```
./mvnw clean package
```

## Run

```
./scripts/run
```

Access the site on the local host at http://localhost:7070

## Deploy

Push to GitHub, wait.
